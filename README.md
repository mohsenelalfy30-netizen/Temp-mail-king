# Temp Mail King 👑

A high-performance, Professional Web Application (PWA) for generating temporary, disposable email addresses. Designed for privacy and speed.

## 🚀 Features
- **Instant Generation**: Get a disposable email address in seconds.
- **Multilingual Support**: Fully translated into 40+ languages.
- **PWA Ready**: Installable on Android, iOS, and Desktop.
- **Privacy Focused**: Protects your real identity from spam and tracking.

## 🛠️ Tech Stack
- **Frontend**: HTML5, CSS3, JavaScript (Vanilla).
- **Hosting**: Vercel & Cloudflare R2.
- **Email System**: Cloudflare Email Routing & Workers.

## 📂 Project Structure
- `/public`: Contains PWA assets, icons, and service worker.
- `/src`: Core logic and multilingual translation files.
- `logic.js`: Handles email fetching and UI updates.
